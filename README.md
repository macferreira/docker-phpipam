# docker-phpipam

Dockerized version of [phpipam](http://phpipam.net) using nginx and php7

Run example:
```bash
$ docker run -p 127.0.0.1:591:80 -e DB_HOST=db-host -e DB_USER=root -e DB_PASS=PaSsWoRd macferreira/docker-phpipam
```
